Пояснения к работе калькулятора

Для переключения знака у текущего числа используйте отдельную кнопку "+\-".

Для очистки дисплея от ошибки или полной очистки памяти калькулятора иcпользуйте кнопку "AC".

Для удаления отдельной цифры текущего числа иcпользуйте кнопку "DEL".

Для возведения в произвольную степень используйте кнопку "pow". Сначала введите первое число, которое будет возводится в степень.Затем нажмите кнопку "pow". После этого введите второе число, которое является степенью первого числа.
"число 1" --> "pow" --> "число 2" => результат

Для вычисления корня с произвольным показателем используйте кнопку "root". Сначала введите первое число, из которого будет извлекаться корень.Затем нажмите кнопку "pow". После этого введите второе число, которое является показателем данного корня.
"число 1" --> "root" --> "число 2" => результат

Ошибки ввода или вычисления выводятся на дисплей калькулятора после полного ввода выражения.
"-9" --> "root" --> " 2" => Ошибка! "2" --> "/" --> " 0" => Бесконечность!

Приятного пользования и тестирования :)
